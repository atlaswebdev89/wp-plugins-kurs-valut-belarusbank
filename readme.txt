Плагин получения курсов валют при помощи API Беларусбанка и отображение их через виджет

== Description ==

Плагин, используя API Беларусбанка, получает текущие курсы валют для Беларуси. Отображение полученных значений осущестсвляется через Виджет.
Использую настройки виджета можно указать валюту, город (города Беларуси) и период обновления.

== Installation ==

1. Скопируйте папку плагина Exchange rate Belarusbank by Atlas в /wp-content/plugins/.
2. Активируйте плагин через меню Плагины.
3. Это все.
